# Projeto API - Filmes
 
## Funcionalidades da API
- A mesma ao iniciar mostra os 20 filmes mais bem avaliados pelos críticos, nela é possível pesquisar por palavras chaves de nomes de filmes e o mesmo mostra os filmes encontrados e é possível entrar para ver os detalhes do filme.

## Nome da API utilizada e URL
- TMDB - The Movie DB.
- URL: https://www.themoviedb.org/

## Tecnologias Utilizadas
- Node
- Vite
- React

## Comandos para instalar dependências
- npm create vite@latest (deve ser escolhido o nome do projeto, após isso escolher a aplicação React e depois Javascript)
- npm install (dentro da pasta principal)
- npm install react-icons
- npm install react-router-dom
